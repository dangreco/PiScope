# PiScope
## Open-source electronics microscope





## Required Materials
---

- 1x Raspberry Pi 3B
- 1x Arduino Mega
- 1x Raspberry Pi Camera
- 1x Extension Cable for Raspberry Pi Camera

## Print Settings
---

- Material Type: PLA
- Layer Height: 0.24mm
- Extruder Temperature: 215
- Bed Temperature: 65
- Brim: 10mm
- Supports: Everywhere
